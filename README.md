# After clone
run followed command
~~~
    npm i
    cd Server
    npm i 
~~~

# When run 
run followed command
1. Open your local mongodb server
2. In root directory run server, if connect succeful, it will show "Connect" in terminal
~~~
    cd Server 
    npm start
~~~
3. Open new terminal in root directroy and run react app
   
~~~
    npm start
~~~
4. Good Luck
